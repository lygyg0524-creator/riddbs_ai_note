ai-note-app 작업 요청 내역
========================================
날짜: 2026-05-09

1. 로그인 화면에 아이디/비밀번호 찾기 기능 추가
   - 로그인 화면 하단에 "아이디 / 비밀번호 찾기" 링크 추가
   - 아이디 찾기: 이메일이 아이디임을 안내하는 메시지 표시
   - 비밀번호 찾기: 이메일 입력 후 Supabase resetPasswordForEmail 호출

2. 프로젝트 재배포 (Vercel 프로덕션)
   - 커밋 후 배포 진행
   - URL: https://ai-note-app-seven.vercel.app

3. 비밀번호 찾기 이메일 미수신 문제 문의
   - Supabase 대시보드 Redirect URL 설정 안내
   - 스팸 폴더 확인 및 무료 플랜 이메일 제한(시간당 4통) 안내

4. 노트 저장 버튼 오류 수정
   - handleSave에 try/catch 추가 → 에러 메시지 화면 표시
   - 새 노트 저장 후 selectedNote 업데이트 → 중복 생성 방지
   - saveNote가 전체 노트 객체 반환하도록 수정

5. 노트 삭제 기능 추가
   - 노트 카드 호버 시 휴지통 아이콘 버튼 표시
   - 삭제 확인 다이얼로그 후 삭제
   - 현재 편집 중인 노트 삭제 시 에디터 초기화

6. 메인 화면 초기 에디터 숨김 + 새 노트 작성 버튼 추가
   - 앱 진입 시 에디터 대신 플레이스홀더 표시
   - NoteList 상단에 "+ 새 페이지" 버튼 추가
   - 노트 선택 또는 새 노트 버튼 클릭 시 에디터 표시

7. 노트 삭제 버튼 디자인 변경
   - 텍스트 "×" → SVG 휴지통 아이콘으로 교체
   - 호버 시 배경색 추가

8. 웹 도메인 변경 방법 문의
   - Vercel 커스텀 도메인 연결 방법 안내
   - 현재 단계에서는 기능 완성 후 도메인 구매 권장

9. Notion 스타일 전면 리디자인
   - 색상 팔레트: #191919 배경, #202020 사이드바, #2383e2 액센트
   - AuthScreen: 언더라인 탭, Notion 스타일 인풋/버튼
   - NoteList: 플랫 리스트 형태 (카드 제거, 📄 아이콘 + 제목)
   - NoteEditor: 대형 제목(text-3xl), 넓은 여백(px-16)
   - Toolbar: Notion 팔레트 적용

10. 아이디 찾기 제거
    - RecoveryScreen에서 아이디 찾기 탭 제거
    - 비밀번호 찾기 단독 화면으로 단순화
    - 로그인 화면 링크 텍스트도 "비밀번호 찾기"로 변경

11. 회원가입 시 닉네임 설정 + 사이드바 닉네임 표시
    - 회원가입 폼에 닉네임 입력 필드 추가 (최대 20자)
    - Supabase user_metadata에 닉네임 저장
    - 사이드바에 이메일 대신 닉네임 표시 (없으면 이메일 폴백)

12. AI 요약 방식 변경: 라이브 자동 요약 → 버튼 클릭 수동 요약
    - 에디터 입력 시 자동 요약 호출 제거
    - SummaryPanel에 "AI 요약 생성" 버튼 추가
    - useGeminiSummary 디바운스 제거, 직접 호출 방식으로 변경
